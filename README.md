# cypress-qe-system

this is a project template for cypress apps.

## Get started

* install the dependencies:

```bash
    cd <your_application_folder>
    npm i
```

* start Cypress

```bash
    npm run cy:run
#or
    npx cypress run
```

*launch eslint

```bash
    npm run lint
```

### work Localy
you need to create a `.env` file on the root folder.
you need to vars : LDAP and PASSWORD
eg.
```bash
    LDAP=<my_ldap>
    PASSWORD=<my_password>
